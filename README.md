# LuckyDraw
B站抽奖工具

## 代码说明
    项目代码采用C#编写
    
    1、采用.net4.8版本，一般情况下win10默认支持。
    2、源码全面升级，做了很多封装，提高效率。
    3、B站接口需要登录，采用官方登录页面。
    4、登录后自动获取up主ID，粉丝自动更新。
    5、定时抽奖更换了抽奖方式。
    6、不支持画册抽奖。
    7、取消接口自动刷新（高频率请求b站会拒绝请求）

新版本需要登录b站，b站很多接口需要用到cookies
